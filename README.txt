Streamlit app for album "recommendations"

=====================================
Contents of directory:

recommender.py
requirements.txt
data (directory)
models (directory)

=====================================

This github repo is used for setting up the web Streamlit app.

https://edavis98-recommender-streamlit-recommender-z6bdeu.streamlitapp.com/

The file 'recommender.py' is the file that gets run.
'requirements.txt' spcifies libraries needed Streamlit needs to run 'recommender.py'

'recommender.py' uses the following to run:
	- data/reviews_w_lang.csv
	- models/knn_model.sav
	- models/tfidf.sav
